https://doc.rust-lang.org/cargo/index.html

### 1. Start

```
$ cargo new bubble_puzzle
```

### 2. Cargo run

> Install missing dependencies & run

```
$ cargo run
```

### 3. Cargo check

> Type check

```
$ cargo check
```

### 4. Cargo build

> Compile & build a binary

```
$ cargo build
```

### 5. Rust docs offline

```
$ rustup doc --std
```
